這是一款專為健身愛好者與飲食控制者設計的輕量化、隱私優先的營養追蹤 Web App。無需註冊帳號，所有數據均儲存於您的瀏覽器本地（LocalStorage），確保數據安全且隨開即用。

🚀 **[立即使用 Live Demo](https://tyrfing2026.github.io/yourdietking/)**

---

## ✨ 功能亮點

### 📊 核心數據視覺化
* **熱量環形追蹤**：即時計算當日剩餘熱量，並以動態進度條呈現。
* **三大營養素指標**：精確追蹤蛋白質、碳水化合物、脂肪的攝取量。

### 🗓️ 智能日曆系統
* **歷史紀錄回顧**：透過自定義日曆插件，快速切換日期查看過去的飲食與飲水。
* **今天導航**：一鍵回到當前日期，確保紀錄不中斷。

### 📑 個人化模板系統
* **動態模板管理**：支援自訂多組營養攝取目標（如：增肌期、減脂期）。
* **即時切換**：下拉選單快速套用目標。

### ⭐ 常用食物清單
* **星號收藏**：點擊紀錄旁的星號即可將常用食物存入清單。
* **快速帶入**：輸入食物名稱時自動過濾常用清單。
