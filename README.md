🥗 飲控助手 (NutriTrack Assistant)
這是一款專為飲食控制愛好者設計的輕量化、隱私優先的營養追蹤工具。無需註冊帳號，所有數據均儲存於您的瀏覽器本地（LocalStorage），確保數據安全且開啟即用。
🚀 立即使用 Live Demo
✨ 功能亮點
• 📊 營養素環形追蹤：視覺化呈現當日熱量與三大營養素（蛋白質、碳水、脂肪）的攝取比例。
• 🗓️ 智能日曆系統：快速切換日期，回顧過去的飲食與飲水紀錄。
• 📑 自訂營養模板：針對「增肌」、「減脂」等不同階段，建立專屬的目標攝取比例。
• ⭐ 常用食物清單：一鍵將明細存為常用食物，大幅提升重複記錄的速度。
• 📱 極致行動端優化：採用響應式設計，在手機上擁有如同原生 App 的操作手感。
• 💾 資料備份與匯入：支援 JSON 格式匯出/匯入，輕鬆遷移數據。
🛠️ 技術棧
• Frontend: HTML5, JavaScript (ES6+), Tailwind CSS
• Icons: Lucide Icons
• Storage: Browser LocalStorage API
• Fonts: Inter & Noto Sans TC
