🥗 飲控助手
這是一款專為健身愛好者與飲食控制者設計的輕量化、隱私優先的營養追蹤 Web App。無需註冊帳號，所有數據均儲存於您的瀏覽器本地（LocalStorage），確保數據安全且隨開即用。

✨ 功能亮點
📊 核心數據視覺化
• 熱量環形追蹤：即時計算當日剩餘熱量，並以動態進度條呈現。
• 三大營養素指標：精確追蹤蛋白質、碳水化合物、脂肪的攝取量。
🗓️ 智能日曆系統
• 歷史紀錄回顧：透過自定義日曆插件，快速切換日期查看過去的飲食與飲水。
• 今天導航：一鍵回到當前日期，確保紀錄不中斷。
📑 個人化模板系統
• 動態模板管理：支援自訂多組營養攝取目標（如：增肌期、減脂期）。
• 即時切換：下拉選單快速套用目標，並支援新增與刪除自訂模板。
⭐ 常用食物清單
• 星號收藏：點擊紀錄旁的星號即可將常用食物存入清單。
• 快速帶入：輸入食物名稱時自動過濾常用清單，大幅節省輸入時間。
📱 頂級操作體驗
• 響應式設計：針對行動端深度優化，擁有如同原生 App 的流暢手感與 UI 佈局。
• 資料備份：支援完整的 JSON 數據匯出與匯入，更換設備也能輕鬆遷移。
🛠️ 技術棧
• 前端開發: HTML5, Modern JavaScript (ES6+)
• 樣式框架: Tailwind CSS
• 圖示庫: Lucide Icons
• 數據存儲: Browser LocalStorage API
• 排版字體: Inter & Noto Sans TC
